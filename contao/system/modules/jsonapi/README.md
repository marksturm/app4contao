# json API für Contao 3.5.x

Dieser Erweiterung stellt eine Json-Schnittstelle für Contao zur Verfügung.

Es werden folgende Inhaltselemente bereitgestellt:

1. Überschrift
1. Text
2. Bild
3. Gallery
4. Liste
5. Youtube
